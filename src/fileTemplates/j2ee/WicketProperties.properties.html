This is a built-in template used by IDEA each time you create a Properties file for a Wicket Page or Panel.
Predefined variables will take the following values:
${PACKAGE_NAME}   name of the package in which the new properties file is created
${NAME}   name of the new properties file
${USER}   current user system login name
${DATE}   current system date
${TIME}   current system time
${YEAR}   current year
${MONTH}   current month
${DAY}   current day of the month
${HOUR}   current hour
${MINUTE}   current minute
